Welcome to USGS-R

The USGS-R repository is a collection of R tools created to retrieve, manipulate, and analyze USGS data. Although the tools have been designed for use with USGS data, many may have utility for other data sets. Tools are created and maintained by USGS scientists, but can be contributed to by any Github user.
