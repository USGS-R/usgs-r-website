GRAN

What is GRAN?

Geological Survey R Archive Network

GRAN is a repository of curated R packages developed by USGS employees for USGS and external use. It allows a user to easily download and install an R package and, in the future, receive updates to that package code as they become available.

The objective of GRAN is to facilitate engagement and R tool sharing across the USGS Water mission area.

How to use GRAN?

There are two ways to take advantage of GRAN tools.

  1. When installing a package, add a 'repo' argument to the install.packages function:

    install.packages("EflowStats", repos="http://owi.usgs.gov")


  2. Alter the Rprofile.site file. This will tell your R environment to always look in the GRAN repository when installing packages and updating packages. This is the preferred method because of the ability to automatically get GRAN updates. To do this, find the Rprofile.site file (in Windows, it is typically in Then add: 
    options(repos=c(getOption('repos'), USGS='http://owi.usgs.gov/R'))`
